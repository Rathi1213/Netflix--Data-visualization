# Netflix-Data-visualization
## Project Overview

This project analyzes the Netflix Titles dataset using Python. It includes data cleaning, exploratory data analysis (EDA), and multiple visualizations to understand Netflix content trends.

## Objectives

- Analyze Netflix Movies and TV Shows
- Visualize content distribution
- Identify top countries producing Netflix content
- Explore release year trends
- Analyze content ratings and genres

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Visual Studio Code

## Dataset

Dataset: netflix_titles.csv

## Visualizations

- Movies vs TV Shows
- Top 10 Countries
- Content Ratings
- Release Year Distribution
- Top Directors
- Top Genres
- Movie Duration Distribution
- Correlation Heatmap

## How to Run

1. Clone or download the project.
2. Install required libraries:

pip install -r requirements.txt

3. Open Data_Visualization.ipynb.
4. Run all cells.
